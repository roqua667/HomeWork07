Command{
1. Преимущества использования паттерна
Паттерн Command Pattern позволяет отделить объект, который вызывает действие, от объекта, который его выполняет.
Преимущества
слабая связность между объектами
возможность хранить команды в истории
возможность реализовать undo/redo
легко добавлять новые команды
}
